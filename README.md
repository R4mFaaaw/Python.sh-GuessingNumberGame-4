# Python.sh-GuessingNumberGame-4
Jadi ini challenge ke 4 untuk saya di website roadmap,sh https://roadmap.sh/projects/number-guessing-game

Penjelasan Fitur
random.randint(1, 100): Digunakan untuk memilih angka rahasia secara acak setiap kali permainan dimulai.

try-except: Digunakan untuk menangani jika user tidak sengaja memasukkan huruf, sehingga program tidak crash.

time.time(): Kita mencatat waktu saat permainan mulai dan saat user berhasil menebak untuk mendapatkan durasi total.

Loop while: Menjaga agar user bisa terus menebak selama sisa nyawa (chances) masih ada.
